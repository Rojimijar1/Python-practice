## CS50 Python Practice

This repository contains my practice exercises and projects completed as part of the **CS50** course by **Harvard University**,
The focus is on Python programming, covering topics such as loops, conditionals, functions, file I/O, data structures, and problem-solving exercises.

## Installation

To run these Python programs, you need to have Python 3 installed. You can install it from [python.org](https://www.python.org/).
You can download my folders and run it in your code editor.
